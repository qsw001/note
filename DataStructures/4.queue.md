# 队列

## 1.1 队列的概念

队列：队列是一种先进先出(FIFO)的线性数据结构，其抽象定义为一个受限的线性表，其操作受限在两端进行。

## 1.2 循环队列

通过使用取模操作，模拟出队列循环的效果，防止溢出。

### 1.2.1 循环队列的基本结构

```
typedef struct queue {
	int data[MAX];
	int rear;
	int front;
}queue;
```

### 1.2.2 循环队列的基本操作

- 初始化队列(initQueue)
    ```
        void initQueue(queue* q) {
            q->front = q->rear = 0;
        }
    ```

- 出队(enqueue)
    ```
        int dequeue(queue* q) {
            if (isEmpty(q)) {
                printf("队列为空\n");
                return 0;
            }
            int x = q->data[q->front];
            q->front = (q->front + 1) % MAX;
            return x;
        }

    ```
- 入队(dequeue)
     ```
        void enqueue(queue* q,int x) {
            //队列满，报错误信息
            if ((q->rear + 1) % MAX == q->front) {
                printf("队列满\n");
                return;
            }
            q->data[q->rear] = x;
            q->rear = (q->rear + 1) % MAX;
        }
    ```
- 取对头元素(getFront)
    ```
        int getFront(queue* q) {
            if (isEmpty(q)) {
                printf("队列为空\n");
                return 0;
            }
            return q->data[q->front];
        }
    ```
  
- 判断队列是否为空(isEmpty)

    ```
        int isEmpty(queue* q) {
            return q->front == q->rear;
        }
    ```
